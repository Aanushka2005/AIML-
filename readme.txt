# Hybrid Intelligent Recommendation & Decision System

## Project Title

Hybrid Intelligent Recommendation & Decision System

---

## Project Overview

This project is an advanced Artificial Intelligence and Machine Learning system designed to recommend movies to users based on their preferences. It integrates multiple AI techniques including Machine Learning, Natural Language Processing (NLP), Search Algorithms, and Probability Theory to simulate a real-world recommendation system similar to platforms like Netflix and Amazon.

The system demonstrates how different AI domains can work together to create a smart, data-driven decision-making model.

---

## Objectives

* Build an intelligent movie recommendation system
* Apply both supervised and unsupervised learning techniques
* Integrate NLP for sentiment analysis
* Demonstrate search algorithms like Breadth-First Search (BFS)
* Apply optimization techniques such as Gradient Descent
* Incorporate probability-based decision-making

---

## Key Features

* User similarity-based recommendations
* User clustering using K-Means algorithm
* Rating prediction using Linear Regression
* Sentiment analysis using NLP (TextBlob)
* BFS traversal for search representation
* Probability-based recommendation scoring
* Gradient Descent optimization for learning

---

## Technologies Used

---

## Dataset

A custom dataset of users and movie ratings is manually created to simulate real-world scenarios.

This ensures:

* Originality of the project
* No dependency on external datasets
* Better understanding of data handling and preprocessing

---

## System Workflow

1. Data Creation and Preprocessing

   * Generate user-movie rating dataset
   * Clean and structure the data

2. Similarity Calculation

   * Compute similarity between users
   * Recommend movies based on similar users

3. Clustering (Unsupervised Learning)

   * Group users using K-Means clustering

4. Prediction (Supervised Learning)

   * Predict ratings using Linear Regression

5. Sentiment Analysis (NLP)

   * Analyze user reviews using TextBlob
   * Classify sentiment as positive or negative

6. Search Algorithm (BFS)

   * Represent recommendation graph
   * Traverse using Breadth-First Search

7. Probability-Based Decision Making

   * Assign scores based on likelihood of user preference

8. Optimization

   * Use Gradient Descent to improve predictions
